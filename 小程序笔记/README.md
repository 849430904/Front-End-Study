### 小程序的特点
* 小程序没有dom对象，一切基于组件化； 组件化：具有特定功能代码、效果的集合（组件包括js html css）

#### 小程序四个重要文件
*  *.js
*  .wxml ->view结构 -->html
*  *.wxss ->view样工 -->css
*  *.json  -->view数据  -->json文件
